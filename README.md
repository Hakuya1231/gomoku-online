# 网页版五子棋（Gomoku）

一个**纯前端**、可离线打开的五子棋小游戏。

## 功能

- 本地双人对战（黑先/白先可选）
- 人机对战（简单/中等/困难 3 档）
- 15×15 / 19×19 棋盘可切换
- 胜负判定（连五即胜）+ 高亮胜利连线
- 悔棋、重开
- 鼠标悬停落子提示、坐标显示开关

## 运行

### 方式 1：直接打开（最简单）

双击打开 `index.html` 即可。

> 如果你的浏览器限制本地文件某些能力（一般不会），用下面的方式 2。

### 方式 2：起一个静态服务器（推荐）

在本目录打开 PowerShell，执行：

```powershell
python -m http.server 5173
```

然后在浏览器访问：`http://localhost:5173`

## 操作

- 鼠标点击棋盘交点落子
- 快捷键：
  - `Z`：悔棋
  - `R`：重开
