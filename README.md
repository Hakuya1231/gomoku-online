# 五子棋在线对战

一个纯前端的五子棋游戏，支持本地双人对战、人机对战和在线联机对战。

## 功能特性

### 游戏模式
- **本地双人** - 黑白双方轮流落子
- **人机对战** - 三档难度（简单/中等/困难），使用 Alpha-Beta 剪枝算法
- **在线联机** - 创建房间分享链接，对手点击即可加入对战
- **观战模式** - 第三人可通过链接观看实时对局

### 游戏功能
- 15×15 / 19×19 棋盘可选
- 胜负判定 + 胜利连线高亮
- 鼠标悬停落子提示
- 坐标显示开关
- 黑先/白先可选

### 联机功能
- 创建房间自动生成分享链接
- 支持一键复制链接
- 对手打开链接自动加入房间
- 观战者可实时观看对局
- 断线提示

## 技术栈

### 前端
- **HTML5 Canvas** - 棋盘渲染
- **原生 JavaScript** - 无框架依赖
- **CSS3** - 深色渐变主题

### AI 算法
- **Minimax + Alpha-Beta 剪枝** - 博弈树搜索
- **启发式评估函数** - 棋型评分
- **搜索深度** - 简单(随机)、中等(4层)、困难(6层)

### 网络通信
- **Firebase Realtime Database** - 实时数据同步
- 无需自建服务器
- 支持 NAT 穿透，跨网络稳定连接

## 运行方式

### 方式 1：直接打开（最简单）

双击 `index.html` 即可运行本地模式（双人、人机）。

### 方式 2：静态服务器（推荐）

```bash
# Python
python -m http.server 5173

# Node.js
npx serve .
```

然后访问 `http://localhost:5173`

### 方式 3：部署到静态托管

可直接部署到 Vercel、Netlify、GitHub Pages 等平台。

> **注意**：联机功能需要 HTTPS 环境（本地 localhost 除外），建议部署到支持 HTTPS 的平台。
